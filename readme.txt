项目开发调试方法：
1.npm install安装依赖
2.gulp dev动态监视更改，如果报错，修改代码后重新gulp dev
3.在chrome里打开build里的main/main.html
4.修改代码在src里想要页面文件夹修改，注意只修改src里的代码，gulp dev会自动打包到build文件夹
5.首页接口在src/home/home.js里（代码注释表明了位置），直接写ajax请求后替换掉setTimeout赋值的模拟代码即可
6.打包可自行eclipse上打包，也可以我帮忙打包
7.其他页面配置如果写好，我可以帮忙整合
8.图标用的自有图标库，可以更换
9.app名称，启动图片自己改好
